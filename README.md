# microX0
مشروع **MicroX** هو غواصة ذكية بهيكل انسيابي كسمكة، صُممت لتنظيف البحار من البلاستيك باستخدام الذكاء الاصطناعي. تعمل بنظام شفط متطور لفصل النفايات وتخزينها، مع توجيه دقيق عبر أنظمة **Arduino/Raspberry Pi**. يمثل المشروع حلًا تقنيًا مبتكرًا وقابلًا للتوسع لحماية البيئة البحرية واستعادة توازن النظام البيئي بكفاءة عالية.
